Maple Valley

Maple Valley was an app built built to help a classmate with registration to the Maple Valley Girl Scout camp. The app is deploid on heroku and uses express, express-handlebars, MySQL, and we used bulma as our CSS Framework.

9-5 Planner

This is a working 9-5 daily planner. It uses Bootstrap as the CSS Framework, and the Moment.js API. You can lock in your entry using the save button or clear using clear. It will change color based on the time of day, Gray is in the past, Red is current, and Green is in the future.

Weather Search

This is a website that will look up the current, and 5 day weather forecast for a city. The previous searchs are saved locally and can be accessed again using the buttons below the search bar. You also have the ability to clear the previous search history using the provided button. This site was built using Bootstrap as the CSS Framework, and the APIs Moment.js and OpenWeather.

Burger-Eater

This is a pretty basic FullStack web app that allows the user to add to a list of burgers that they would like to eat and then move them to the eatin list after consumption. This was made using express, express-handlebars, MySQL.
